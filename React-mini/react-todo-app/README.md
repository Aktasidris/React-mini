# 📝 React Todo App (Gelişmiş)

## 📄 Açıklama
Karmaşık yapılarda görev yönetimi için yapılmış gelişmiş Todo App.

## 🚀 Kullanılan Teknolojiler
- React
- Tailwind CSS
- LocalStorage

## 📦 Projeyi Çalıştırma
Terminal üzerinden aşağıdaki komutları çalıştırarak uygulamayı başlatabilirsiniz:

```bash
cd react-todo-app
npm install
npm run dev```
